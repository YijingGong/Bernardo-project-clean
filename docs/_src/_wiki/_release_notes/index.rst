Release Notes
=============

This section documents changes to RuFaS across versions, including new features,
bug fixes, data and I/O updates, and internal tooling changes.

Versions follow semantic versioning (MAJOR.MINOR.PATCH). Major versions may
introduce breaking changes.

For details on versioning rules and release process, see:
:doc:`../_wiki/RuFaS-Versioning-Policy`

Structure
---------

Each release note is organized into the following sections when applicable:

- Highlights
- Scientific Evaluation
- Improvements
- Bug Fixes
- Data / I/O Changes
- Internal / Tooling
